## Installation
Add this line to your application's Gemfile:

```ruby
gem 'panicboat', path: '/panicboat'
```
or
```ruby
gem 'panicboat', git: 'https://github.com/panicboat/api-plugin'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install panicboat
```
